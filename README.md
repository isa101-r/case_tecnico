# case_tecnico


Este projeto consolida e analisa os preços do cobre no período de 2020 a 2025.

Foram realizados:
- Consolidação dos arquivos originais (sem alterações manuais)
- Cálculo de médias e variações mensais, trimestrais e anuais
- Construção de gráficos
- Análise sintética da trajetória dos preços

## Estrutura

- `case_tecnico.Rmd` → Código reproduzível
- `case_tecnico.docx` → Versão em Word (knit)
- `lme_prices (1)` → Dados utilizados

## Como rodar

Abra `case_tecnico.Rmd` no RStudio e execute todos os chunks ou clique em **Knit**.  
Os resultados são gerados automaticamente a partir dos arquivos originais.
